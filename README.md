# 🏨 Sistema de Gestão de Hospedagem em Hotel

> **Status do Projeto:** Concluído ✔️

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido para consolidar conhecimentos avançados em C# e .NET, focando na manipulação de objetos, coleções e lógica de negócios. O sistema gerencia a reserva de suítes, calculando o valor total com base na quantidade de hóspedes e dias de permanência, aplicando descontos para estadias longas.

## 🛠️ Tecnologias Utilizadas
* **C#**
* **.NET 6.0+**
* Programação Orientada a Objetos (POO)

## 🧠 O que eu aprendi
* Implementação de regras de negócio complexas (ex: validação de capacidade da suíte).
* Cálculo dinâmico de valores com aplicação de descontos.
* Organização de código em classes e métodos reutilizáveis.
* Uso de diagramas de classe para planejamento de software.

## 🚀 Como executar o projeto localmente
1. Clone o repositório: `git clone https://github.com/meloalexandev/trilha-net-explorando-desafio.git`
2. Navegue até a pasta do projeto.
3. Execute o comando: `dotnet run`
